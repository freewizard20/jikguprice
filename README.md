# [jikguprice](https://jikguprice.com)
## version 1.3

> Serverless SPA web application on foreign shopping calculations


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
